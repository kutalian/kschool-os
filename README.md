# kschool-os

A comprehensive School Management System built with Laravel.

## Features

- **Role-Based Access Control**: Admin, Staff, Student, Parent.
- **Academic Management**: Classes, Subjects, Exams, Grading.
- **Attendance System**: Daily student and staff attendance.
- **Reporting**: Auto-generated report cards and admit cards.
- **User Management**: Detailed profiles and role management.

## Installation

1. Clone the repository.
2. Run `composer install`.
3. Copy `.env.example` to `.env` and configure database.
4. Run `php artisan migrate --seed`.
5. Serve with `php artisan serve`.
